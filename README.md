# Child Nutrition and IFA Analysis using NFHS-5 Data

## Project Overview

This project analyzes cleaned NFHS-5 child-level data using Python and Google Colab. The main objective is to study selected child health and nutrition outcome indicators and compare them across treatment groups.


## Dataset

The analysis is based on cleaned NFHS-5 child-level data.

The original dataset is not uploaded in this repository due to data-sharing and file-size restrictions.

## Objectives

- To analyze selected child health and nutrition outcome indicators
- To compare outcomes between treatment groups
- To apply survey weights for weighted percentage calculation
- To generate healthcare data insights using Python
- To create a beginner-friendly data analytics portfolio project

## Methodology

The analysis was performed in Google Colab using Python.

Main steps included:

1. Importing the cleaned dataset
2. Checking the dataset structure
3. Identifying treatment and outcome variables
4. Applying survey weights
5. Calculating weighted percentages
6. Comparing child outcome indicators by treatment group
7. Generating visual outputs for interpretation
